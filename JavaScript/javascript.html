Question 1

Create a function to get the grade from a quiz object

            
                function getGrade(quiz)
                {
                    return quiz.grade;
                }

The function above takes a specific penguin's quiz object (passed in as 'quiz') and returns the grade va.

Example of how you would call the function:

            
                    getGrade(classroom[0].quizes[0]);

Question 2

Find the mean of the quiz grades for the first penguin

            
                var vals = classroom[0].quizes.map(getGrade);
                d3.mean(vals);

The first line creates a variable with the identifier 'vals' and a value equal to the array of quiz grades created from the map function. Map takes an object and transforms it to an array.

The second line uses the d3 library's mean function to find the mean of the values in the array passed in to it (in this case the array vals).

Question 3

Create a function to find the mean of the quiz grades for a given penguin

            
                var gradeMeans = function(peng)
	            {
		           var quizzes = peng.quizes.map(getGrade);
		           return d3.mean(quizzes);
	            }

This function takes in a penguin object as its parameter. The variable quizzes holds the array of that specific penguin's quiz scores. The last line returns the mean of the quizzes array.

Question 4

Create an array of the mean quiz score of each penguin. Use one line of “code”

        
                    classroom.map(gradeMeans);

This line takes the array 'classroom' and maps it to an array using the function gradeMeans to find the mean quiz scores for all the penguins.

Fun fact: we originally tried to make it more complicated than it actually was by trying to use a forEach function.
