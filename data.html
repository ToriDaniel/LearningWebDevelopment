Norms and Exceptions

Norm
Also called the smooth
The values that closely relate to one another
Exceptions
The values that skew the data due to the fact that they are widely different from the other values in the set

To determine what values fall into the norm and whether we have any exceptions, we must take use a few techniques to view the patterns and trends of the data.

The Distribution is a term used to describe all of the values in a category.

"A visualization designer should never rely on a single statistic or a single chart or map when doing exploratory work," (Cairo, ch.6)

Important terms to know

Measures of central tendency
gives the level of distribution or the average size of numbers and their center point.
Parameter
a number that summarizes or describes an entire population
Statistic
the term used to describe when just a sample of data is taken and the same parameter is found

Mode

value that appears most often

Unimode is a distribution that only has one mode. If there are more than one then they would be called bimodal, trimodal… and so on depending on how many modes there are.

Median

The middle value of the data when in order from smallest to largest

This is also resistant to outliers because it would be unaffected. For example, if you added on an outlier to the end of a large set of vales, the middle number would either go up to the next number or to the middle of two values.

Mean

(the average) adds all the data together, then divides it by how many values there are.

Range

The difference between the largest and smallest values in a data set

Outliers

values that are so far out of the norm that they throw off measure of central tendency if they are not addressed

if mean and median differ by a large margin, this means that there is something in the data, such as an outlier, that needs to be addressed through means such as weighting.

Histograms

A common way to display these measures of central tendency is to use a histogram. Histograms are a visualization of data that shows bars to represent the frequency (or how many instances) of each value we have in our data. This allows us to see the shape of our data.

Weighting

To calculate the mean of groups of data, and the groups are of different sizes, you need to perform a different calculation

This ensures that smaller groups are not calculated using the same weight as bigger groups.

What is correlation used for?

It is used to talk about the association between data and how closely they are related using the correlation coefficient.

What is the difference between correlation and causation?

Correlation means that when one variable changes, the other one also changes, but the change in variable x is not directly responsible for the change in y.

Standard Deviation and Percentiles

  • There are four elements of Distribution: Level, Spread, Shape, Outliers
  • You can use standard deviation or percentiles to give numerical summaries of these distributions. Giving just the average doesn’t give the whole story.
  • Standard deviation lets the viewer know how much the data deviates from the mean.
  • Percentiles break up the data to tell where the data falls on the line between the smallest and largest value in the data. This can also be represented through box plots.

Visual Representations

  • Frequency charts are an alternative to histograms and it shows the range of the data as well as the modes.
  • To give a more complete story, it is also possible to use more than one visualization to present the data.
  • When looking at visualizations of data, it can be difficult to remember that data is also dependant on population size. Small populations will have more variation than large populations.
  • A visualization that takes this into account is a funnel plot, which plots the data in relation to the population size.
