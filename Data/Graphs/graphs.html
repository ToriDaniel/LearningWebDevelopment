Graphs

Ordinal Scales

Handles the left and right positioning of bars and labels along the x-axis.Ordinal scales are used for categories that have some form of order to them. This ensures that when updating the graph, the bars remain in the same order.

Bands

Bands ensure that when data is grouped together for display, they are banded together evenly.

How to update data

For updating data where the data values are all updated and the number of values remain the same:

  • Modify the values in the dataset
  • Rebind the new values to the existing elements
  • Set new attribute values to update the visual display

For this update to happen, there needs to be some sort of "trigger" or event, such as clicking on a button.
